# Starting Component Test Explorer

## About this task

## Procedure

1. In the Servers view, ensure that your server is started.
2. Right-click the server and select Launch > Component
Test Explorer. (Alternatively, you can start Component
Test Explorer from an external web browser using the URL http://hostName[:port]/ComponentTestExplorer, where hostName is the hostname of your machine and port is the HTTP port
of your server).
3. If the Component Test Explorer application is not yet deployed
on the server and a window opens that asks if you want to deploy
it, click Yes. When another window opens
to confirm that Component Test Explorer has been successfully installed,
click OK in the window. The Log In page opens.
4. In the Log In page, specify your user name and password.
5. Click LOGIN to log in to Component Test Explorer.
