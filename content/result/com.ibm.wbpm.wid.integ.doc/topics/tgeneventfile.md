## About this task

## Procedure

1. In the Business Integration view, select the mediation
flow that contains an Event Emitter Primitive.
2. Right click the mediation flow and select Monitor
Tools -> Generate Event Definitions. An
event definition is generated for each Event Emitter primitive in
the mediation flow. The event is given the name of the Event Emitter's
label property. In the Physical Resources view ,
the event definition is generated (or regenerated) into the events
