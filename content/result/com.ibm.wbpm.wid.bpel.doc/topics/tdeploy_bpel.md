# Deploying a BPEL process

- Planning to deploy a process

A list of considerations to keep in mind as you plan your deployment is presented.
- Deploying a BPEL process to the IBM Workflow Server

You can distribute Service Component Architecture (SCA) modules that contain BPEL processes or human tasks, or both, to deployment targets. A deployment target can be any server or a cluster that has Business Process Choreographer configured on it.
