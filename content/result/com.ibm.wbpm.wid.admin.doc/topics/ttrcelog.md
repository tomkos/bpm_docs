# Enabling debug trace logging in the integration debugger

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select Debug. The Debug page opens.
3. Select the Enable trace log check
box.
4. Click OK. From this point on, the
integration debugger debug trace will be logged in the widv6debug.log
file and the .log file in the .metadata folder.
