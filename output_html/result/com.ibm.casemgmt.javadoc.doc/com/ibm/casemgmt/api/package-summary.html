Package com.ibm.casemgmt.api

See: Description

  • Class Summary 

Class Description

Case

Represents a case in the case management system.

Case.FetchHistoryResult

Represents information returned from fetching case history.

Case.RelationshipResult

Represents information returned from creating a relationship.

CaseMgmtObjectStore

Represents an object store that contains a deployed solution in the Case Management system.

CaseMgmtObjectStore.CM8HostConfiguration

Represents information about the CM8 host configured for this object store.

CaseMgmtUpdatingBatch

Submits changes to Case Management objects in a batch.

CaseRelationship

Represents a relationship between two cases.

CaseType

Represents a case type which is part of a deployed solution in the Case Management system.

CaseType.FetchDynamicTasksResult

Represents information returned from fetching dynamic tasks.

Comment

Represents a comment applied to a case.

DeployedSolution

Represents a deployed solution in the case management system.

DocumentType

Represents a document type deployed as part of a deployed solution in the Case Management system.

ExternalDocumentReference

Represents a reference to an external document filed in a Case folder or a sub-folder of a Case.

HealthAlert

Represents a JSON object that is stored in the CmAcmHealthDetails property of the Health Condition object.

HealthCondition

Represents a HealthCondition instance that is associated with a case.

Stage

Represents a case stage in IBM Case Manager.
