# Moving extended data elements up or down in event definitions

In the event definition editor, you can move extended data
elements up or down in your event definitions without needing to directly
modify the underlying source code.

## About this task

## Procedure

1. In the event definition editor, select the extended data
element that you want to move up or down.
2 Complete one of the following steps:
    - If you want to move the extended data element up one position,
right-click the extended data element and select Move Up.
The extended data element moves up one position in the list of extended
data elements.
    - If you want to move the extended data element down one position,
right-click the extended data element and select Move Down.
The extended data element moves down one position in the list of extended
data elements.
