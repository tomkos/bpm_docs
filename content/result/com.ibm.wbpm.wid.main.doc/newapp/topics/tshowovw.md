# Hiding or displaying integration solution diagram overviews

An integration solution diagram overview is a small image
of the entire integration solution that is displayed in the lower
right corner of the integration solution editor. It enables you to
see the structure and content of the integration solution without
scrolling. You can hide or display the overview as necessary.

## About this task

## Procedure

1. If you want to display the overview, click the Show
Overview icon .
The integration solution diagram overview is displayed and the icon
toggles to the Hide Overview state.
2. If you want to hide the overview, click the Hide
Overview icon .
The overview is hidden from view.
