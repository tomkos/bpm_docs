# Coloring modules

In the integration solution editor, you can color modules
or change the color of modules that are already colored. This enables
you to more easily distinguish one module from another or perhaps
assign certain colors to groups of modules.

## About this task

## Procedure

1. In the integration solution editor, click the module that
you want to color.
2. Click the Change Color of Selected Modules icon . The color
palette opens.
3. In the color palette, click the color that you want to
use for the selected module. The color palette closes and the module
changes to the selected color.
