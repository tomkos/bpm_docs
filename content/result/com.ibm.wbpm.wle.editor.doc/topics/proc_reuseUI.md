# Making process instance user interfaces reusable

## Before you begin

## About this task

## Procedure

1. Open the generated client-side human service that you want to reuse.
2. Switch to Overview, and click Make the UI
reusable.
3. Make the changes that you want to your service variables
and diagram.
4. Open the process and switch to Views.
5. Select the type of UI that you are creating, for example
Details UI > Default or Launch UI > Default. A Launch UI service is
available only if the Start Event for your process is set to None.
6. Click Select....  The
list of selectable services appears, showing the service that you
made reusable.
7. Select the service, and map the data in the Input
Mapping or Output Mapping sections.
8. Click Save or Finish Editing.
