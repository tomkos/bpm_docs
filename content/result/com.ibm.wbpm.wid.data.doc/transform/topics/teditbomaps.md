# Editing business object maps in Integration Designer

- Editing business object maps using the graph view

You can edit business object maps using the graph view in the business object map editor. Note that the graph view is the default view.
- Editing business object maps using the table view

You can edit business object maps using the table view in the business object map editor.
- Editing business object maps using the Properties view

You can edit business object maps using the Properties view in the business object map editor.
