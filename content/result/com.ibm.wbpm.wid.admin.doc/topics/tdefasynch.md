# Defining invocations as asynchronous

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
invocation that you want to define as asynchronous.
3. In the Detailed Properties area, select the Asynchronous
mode check box. The invocation is now defined as asynchronous.
