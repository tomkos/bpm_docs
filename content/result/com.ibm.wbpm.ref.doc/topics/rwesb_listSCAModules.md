# listSCAModules command

This command lists all the SCA modules that have been
deployed to the cell and the applications or process applications
they are associated with.

## Location

Start the wsadmin scripting client from the install\_root/profiles/deployment\_manager\_profile/bin directory.

## Syntax

```
listSCAModules
```

## Parameters

None.

## Example

```
AdminTask.listSCAModules()
```
