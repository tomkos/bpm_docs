# Enabling JavaScript debugging for coaches

## Before you begin

## About this task

## Procedure

1. Open the administrative console and click Resources > Resource Environment > Resource Environment Provider
2. On the Resource environment providers page, click
Mashups\_ConfigService.
3. Under Additional Properties, click Custom properties.
The list of custom properties opens.
4. Click isDebug, change the Value field to
true, and then click OK.
5. Save your changes to the master configuration.
6. Restart the application server instance.
