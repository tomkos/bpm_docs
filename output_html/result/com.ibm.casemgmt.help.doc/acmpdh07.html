Changing primary queue definitions after upgrade

About this task

Procedure

To change the primary queue definition of a role:

  1. In a stand-alone IBM FileNet Process Designer, open the solution for editing the FileNet P8 activity workflow.
  2. Click Views > Roles.
  3. Select the role name and open the Custom Attributes page to view the value set for the CB_PrimaryQueue attribute.
  4. Enter an existing queue name.
  5. Save and close the solution in IBM FileNet Process Designer.
