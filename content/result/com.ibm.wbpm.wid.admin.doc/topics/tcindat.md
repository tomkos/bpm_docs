# Changing variable or message values in the integration debugger

## Before you begin

## About this task

To change variable or message values:

## Procedure

1. In the Debug view, ensure that the correct stack frame
is selected for the variable or message.
2. In the Variables view, locate the variable part or message
element that has the value you want to change.
3. Right-click the variable part or message element and select Change
Value. The Set Value window opens.
4. Type the new value into the text area and click OK.
