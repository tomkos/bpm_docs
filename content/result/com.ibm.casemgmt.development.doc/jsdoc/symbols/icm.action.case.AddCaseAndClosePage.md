# Class icm.action.case.AddCaseAndClosePage

Extends
					icm.action.Action.

				
			
				Saves the case that is being added and then closes the current Add Case page or 
       Split Case page.
       
       Context required by this action: [['NewCase', 'Coordination']]

       This action starts the following series of topics of coordination in sequence:

 'COMMIT'
 'VALIDATE' 
 'BEFORESAVE' 
 'SAVE'
 'AFTERSAVE' 

       The coordination is started with the following context:

 'FORCASE': true
