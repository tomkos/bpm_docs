## Configuration properties

| Configuration property   | Configuration variable   | Description                                                                                                                                                                            |
|--------------------------|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Sidebar width            | sidebarWidth (Integer)   | The width of the sidebar section in pixels. You can specify a width as a number (pixels are assumed), a number with px as the unit of measurement, or a percentage.Default: 315 pixels |
