Interface UnmodifiableProperty

  • public interface UnmodifiableProperty Represents a non-modifiable interface to a property. This interface allows the value and other attributes of the property to be accessed but the value cannot be modified.

    This interface can be obtained from a Case Manager Java API object, such as a Case object, when there is no intent to modify the properties of the object. In that scenario some communication to the server can be optimized. In particular, communication with an external data service can be deferred until there is an intent to modify the object.

    Most of the methods on this interface correspond to non-modifying methods on the CaseMgmtProperty class. Some methods on that class are only applicable when communication with an external data service has occurred, such as the hasDependentProperties method.
    Such communication does not occur until there is an intent to modify the properties of the object. Therefore those methods are not included on this interface.
