# Creating sticky notes

## About this task

## Procedure

1 In the Business Integration perspective, complete one ofthe following steps: A sticky note displays and automatically connectsto the artifact.
    - Click the artifact that you want to attach a sticky note to. The
sticky note icon () shows
in the artifact pop-up menu.
    - Right-click inside the artifact and select Add Note.
    - Click the sticky note icon in the palette, and then move the cursor
to the artifact that you want to attach the sticky note to.
2. Click inside the sticky note and enter your text.
3. Press Ctrl+S to save your work.
