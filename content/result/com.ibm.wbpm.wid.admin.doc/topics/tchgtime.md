# Changing integration debugger timeout values

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the left frame of the Preferences window, expand Java and
select Debug. The Debug page opens.
3. In the Debugger timeout (ms) field,
ensure that the value is at least 300000 milliseconds,
which is five minutes. You can increase the debugger timeout value
as needed.
4. In the Launch timeout (ms) field,
ensure that the value is at least 20000 milliseconds.
You can increase the launch timeout value as needed.
5. Click OK.
