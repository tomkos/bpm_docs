public final class SimpleVWSessionCache extends BaseVWSessionCache A basic implementation of VWSessionCache that is appropriate to cache the VWSession for a single user in the context of a thread of execution. It caches VWSession objects in a map of this instance so it dedicated to the current calling thread.
