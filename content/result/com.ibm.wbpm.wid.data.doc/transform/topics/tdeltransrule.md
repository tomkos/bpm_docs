# Deleting transforms

## About this task

- From the graph view, you can:
    - right-click the transform and select delete from
the menu.
    - select the transform and press the delete key.
    - select the transform and use the delete button
on the toolbar.
- From the table view, you can:

- select the transform and use the delete button
or the delete key.

You can select multiple transforms using CTRL
+ click or marquee selection. With multiple transforms,
the same deletion steps can be used.
