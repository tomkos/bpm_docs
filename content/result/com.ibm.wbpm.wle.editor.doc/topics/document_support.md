# Enabling document support

You can use a document management system to store and interact with documents that you
use in your process. You can create, query, view, update and delete documents.

- Working with BPM documents

You can use Enterprise Content Management (ECM) tools to work with BPM documents in the embedded BPM document store. For example, you can create, edit, and work with documents in the document store using either a coach or a heritage coach. There are, however, special considerations for some of the CMIS operations that can be performed.
- Integrating with Enterprise Content Management (ECM) systems

You can interact with an Enterprise Content Management (ECM) server to store or view documents.
