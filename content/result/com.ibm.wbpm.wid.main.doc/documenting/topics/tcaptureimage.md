# Capturing images

You can export artifacts from your workspace as images and save
them in the file system.

## About this task

You can export many artifacts as images, including the assembly
diagram, business objects, business processes, human tasks, and others. If Export
as Image is enabled on its pop-up menu, you can capture the artifact
as an image.

To generate and save an image, follow these
steps:

## Procedure

1. In the Business Integration view, select the artifact that you
want to show as an image.
2. Right-click the artifact and select Export as Image from
the pop-up menu.
3. In the Export as Image wizard, select the file format that you
want to use and the location where you want to save the file. Supported
image formats are SVG, JPEG, PNG, and TIFF.
4. Click Finish to generate the image file
in the location that you specified.
