# Adding monitors

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. Click the Add icon . The New Configuration wizard opens.
3. In the Wizards list, select Monitor and
click Next. The Select a Module page opens.
4. In the Select a module to continue list,
locate the test configuration that contains the module where you want
to add one or more monitors, then select the module.
5. Click Next. The Add Monitors page
opens.
6. In the Monitored wires and exports in this module list,
ensure that the check box is selected beside the name of each wire
and export that you want to monitor.
7. Click Finish. The monitors are added
to the selected test configuration module.
