# Reserved error codes

The reserved error codes are provided by database vendors and other vendors and are subject to
change by the vendors. However, the current reserved error codes are shown in the following
list:

- errorcode=-4498
- errorcode=-4499
- sqlcode: -4498
- sqlcode: -4499
- sqlcode: -911
- sqlcode: -913
- sqlcode=-911
- sqlcode=-913
- ora-00060
- ora-02049
- fnrcd0009
- fnrce0019
- fnrcm0034
- metadata\_stale\_cache
- deadlock
