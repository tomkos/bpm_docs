getCaseStructure operation

| Parameter | Type | Description | |--------------|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------| | caseId | String | The identifier of the case. You must set the expression for the caseID parameter to the F_CaseFolder business object field in the Expression Builder window. | | return_value | String[] | The CaseStructure value is an array of strings, containing the case structure as strings of the form folder_path/=doc_version_series_ID. |
