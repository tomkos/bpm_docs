# Editing business object maps using the Properties view

## About this task

## Procedure

1. Choose an element in the business object map and edit the
properties (if the fields can be edited and are available) in the
Properties view. Click the transform. For example, you can select
the Submap transform.
2. In the Properties view you will see the elements of that
transform, including Description, Details, and Sources/Targets.
Click each of these options to edit the properties where applicable.
(Note that the available options will differ depending on the business
object map item selected.)
