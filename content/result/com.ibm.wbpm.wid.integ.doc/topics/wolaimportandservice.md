# Creating a WOLA import and service

## Before you begin

## Procedure

To create a WOLA import and service, complete the following
steps:

1. Select File >  New >  External Service. Alternatively,
you could open the Assembly editor and drag a WOLA import component
onto the canvas. The New External Service wizard
opens.
2. From the available types, select Optimized Local
Adapter for z/OS and click Next.
The Select an Adapter page opens.
3. Select WebSphere Optimized Local Adapter.
When you select WebSphere Optimized Local Adapter,
the minimum version of the RAR file, version 2, is also selected.
This RAR file comes from WebSphereÂ® Application Server
for z/OS 8.0.0.3.
4. Continue through the wizard.
