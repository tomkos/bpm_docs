# Overview of JMS function selectors

Some function selectors are supplied. In the default function
selector section, we discuss the general behavior of a function selector and
the implementation of the default function selector. In the standard JMS message
function selector section, we discuss a particular function selector for the
body types used by this standard.

- Default JMS function selector

The default JMS function selector is described.
- JMS function selector

The JMS function selector for the JMS standard of defining a message class with subclasses for the payload is discussed.
