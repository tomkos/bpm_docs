# Deleting exports and imports

## Procedure

To delete exports and imports, complete the following
steps:

1. In the assembly editor, right-click the export or import
that you want to delete and select Delete.
A Confirm Delete window opens.
2. If you are deleting a web service export and it has a WSDL
port that does not exist in the same file as a WSDL port type, select
the check box that is displayed in the window to delete the WSDL
file that contains the port.
3. Click Yes.
