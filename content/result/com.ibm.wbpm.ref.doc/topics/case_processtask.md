# Process tasks

The Case tasks list view lists all the process tasks that are associated with the
configured case instance. It doesn't list completed tasks.

- Filter the process tasks to view the tasks only in the case tasks list.
- Expand process task to view the associated task properties along with the business data.
- Claim, update, and reassign tasks.
- Use the Modify button to perform batch operations on
Priority, Assignee, and Due
date for multiple process tasks.
