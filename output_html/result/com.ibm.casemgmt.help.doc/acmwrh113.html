Add Activity pages

Two default Add Activity pages are provided:

  • Add Activity
  • Add Activity Form

Add Activity page

  • The Work Item Toolbar widget includes the following buttons: Add and Cancel.
  • The Case Information widget displays the following tabbed views: Documents, History, and Activities.

Add Activity Form page

  • The Work Item Toolbar widget includes the following buttons: Add and Cancel.
  • The Case Information widget displays the following tabbed views: Documents, History, and Activities.
