# Adding HTML meta tags to client-side human services

## About this task

## Procedure

To add an HTML meta tag to your client-side human service:

1. Open an existing client-side human service, or click the plus sign next to User
Interface to create one.
2. Switch to Overview and, under HTML Meta Tags,
select the meta tag that you want to add, and enter the corresponding tag content.
3. In the list of available meta tags, select the check box to turn on a meta tag or clear
the check box to turn off the meta tag.
4. Click Save or Finish Editing.
