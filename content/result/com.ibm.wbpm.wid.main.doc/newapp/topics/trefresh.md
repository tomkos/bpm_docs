# Replacing an implementation

If you have made changes to components in the assembly diagram,
you can replace existing implementations of components with new ones.

## About this task

## Procedure

1. In the assembly editor, click the implemented component so that
it is selected on the canvas.
2. Right-click and select Regenerate Implementation from
the pop-up menu. You will be prompted to confirm that you want to replace
the implementation files. Click OK to continue.
The implementation is opened for editing.
