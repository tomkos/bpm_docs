# Setting preferences for the test suite editor

## About this task

- Specifying the maximum depth of expanded business objects

In the test suite editor, nested business objects are automatically expanded to a depth of five levels in the test data table. However, you can change the maximum depth to which you want to automatically expand your nested business objects.
- Customizing keyboard shortcuts

In the test suite editor, there are several actions that you can perform using only keyboard shortcuts. If you want, you can use a preference to customize some of the keyboard shortcuts.
