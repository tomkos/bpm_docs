Debugging events by using the Script Adapter widget

Procedure

  1. In Page Designer, select Edit Settings from Script Adapter widget menu.
  2. Select one or both of the following check boxes:

Show Script Text Select this check box to display the text of the script as the script is being run. Block Outbound Event Select this check box to prevent the outgoing event from being sent by the Script Adapter widget. If you have wired the Send event payload event to another widget, you can select this option to temporarily stop the Script Adapter widget from sending the event while you are debugging the script. 3. Click OK.
