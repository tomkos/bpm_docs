  • public class CloseCase extends java.lang.Object Represents the CloseCase class. This class has the methods to delete the case artifacts. To instantiate the class, use the constructor CloseCase by passing the parameters in JSON.

After instantiating the class, call the method validateParams to validate the parameters.

After getting a successful string result from the validateParams method, call the deleteCase method to delete the case artifacts
