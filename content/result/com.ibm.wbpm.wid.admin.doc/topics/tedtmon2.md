# Editing monitors

## About this task

## Procedure

1. In the test suite editor, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration,
module, and Monitors containers.
3. In the Monitors container, select
the monitor that you want to edit.
4 In the Detailed Properties area,complete one or both of the following tasks:
    - If you want the selected monitor to monitor requests, select
the Monitor requests check box.
    - If you want the selected monitor to monitor responses, select
the Monitor responses check box.
