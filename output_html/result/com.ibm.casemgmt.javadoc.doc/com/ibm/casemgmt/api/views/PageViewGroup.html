Class PageViewGroup

  • java.lang.Object

    • com.ibm.casemgmt.api.views.PageViewGroup

  • public final class PageViewGroup extends java.lang.Object Represents a page view group for a case type. Page views are defined by the solution to control how the case properties are displayed in different types of views. If the view has subgroups of properties, the information about those groups is returned as instances of this class.
