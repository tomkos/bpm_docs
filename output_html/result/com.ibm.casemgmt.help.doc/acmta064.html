Causes

The configured system user that is used for online deployment has not been added to the security manifest.

Resolving the problem

  1. On the workflow server, launch the IBM Content Navigator bawadmin desktop.
  2. In the security manifest, add the configured system user (e.g. ECMoC_Service_Account).
  3. In Workflow Center, deploy another solution snapshot online to the workflow server.
