## Restrictions and limitations

None

## Data binding

| Binding description                                  | Data type   |
|------------------------------------------------------|-------------|
| Records whether the user clicks the button instance. | Boolean     |

## Configuration properties

| Configuration property   | Description                                                                                                                                                                                                                                                | Data type   |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------|
| Allow Multiple Clicks    | Select this option if the user can repeat the button action, such as adding something for each click. Do not select this option if the event is to occur only once, such as a bill payment confirmation.The default value is only allow one click (False). | Boolean     |
