# Adding invocations

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
test case for which you want to create an invocation.
3. Click the down arrow beside the Add Test Case icon  and select Add Invocation.
A new invocation is displayed under the selected test case in the
Test Cases area.
