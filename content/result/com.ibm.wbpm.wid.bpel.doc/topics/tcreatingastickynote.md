# Creating task tags

## About this task

To create
a task tag, complete the following steps:

## Procedure

1. In the Javaâ„¢ perspective
select Window > Preferences.  The Java perspective Preferences window
opens.
2. Expand Java > Compiler and click Task
Tags. The Task Tags window
opens.
3. Click New. 
The New Task Tag window opens.
4. Enter a name for the new task tag in the Tag field.
5. Select a priority level in the Priority field.
6. Click Ok. The Task
Tags window shows the new task tag.
