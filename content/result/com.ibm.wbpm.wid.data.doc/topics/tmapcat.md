# Organizing data maps in the data map catalog

## Before you begin

## About this task

## Procedure

1. To open the data map catalog: from the Business Integration
view, select a project, right click and select Open Data
Map Catalog
2. You can also open the catalog from the toolbar by clicking
the Open Data Map Catalog icon
3. In the catalog, you can set the scope to a single project
or to the workspace. This scope determines the maps that you see in
the catalog.
4. You can filter maps, and open maps from the catalog to
edit them.
5. The map catalog lists the input and output business objects
of each map, so that if you have a large number of maps in your workspace,
you can easily locate the map that you want by searching for the business
objects associated with it.
