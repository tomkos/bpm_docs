# Specifying experts for an activity

## About this task

- Users who have completed this activity in the past, based on historical
analysis. This list is limited to a small group of users who have
completed the activity most frequently.
- Users belonging to a team that is explicitly specified in Process
Designer as an expert group for this activity.

## Procedure

1. Open the process.
2. To explicitly specify an expert group for an activity,
select the activity and go to the Assignments tab
in the Properties view.
3. Specify the relevant team in the Experts Team field.
If you have not already created a team that defines the experts for
this task, you can create a new team to use.
4. Click Save or Finish Editing.
