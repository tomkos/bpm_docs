# Changing deployment properties

## About this task

## Procedure

1. In the Business Integration view, right-click your module
and select Properties. The Properties window
box opens.
2. In the left frame of the window, select Integration
Test Client. The Integration Test Client page opens.
3. In the Integration Test Client page, change the deployment
location or its properties. (If you need additional information about
selecting deployment locations and properties, see the topic "Deploying
modules to the server.")
4. Click OK to close the Properties
window.
