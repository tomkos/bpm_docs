# Enabling prompting when duplicate files are encountered in
the integration debugger

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select Debug. The Debug page opens.
3. Select the Always prompt for file selection
if there are duplicate files found in workbench check
box.
4. Click OK.
