# Examples of files used in batch component testing

- Example of an Ant script

An Ant script is used to perform several tasks, such as building and deploying the component test projects.
- Example of an XSD file

An XML schema definition (XSD) file is used to define the contents of the XML output file generated from a test run.
- Example of an XML output file

When you run an Ant script, an XML output file is generated.
