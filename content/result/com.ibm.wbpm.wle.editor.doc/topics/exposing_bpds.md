# Exposing processes

## About this task

- Start instances of the process in Workplace or Process Portal.
- View process instance data in reports in Process Portal.

Expose a process to particular teams to establish who can start instances of the
process in Workplace

## Procedure

1. Open the process.
2. Click the Overview tab.
3. In the Exposing section, configure the exposure settings. 
Table 1. Settings that can be enabled in the Exposing
section

Exposure setting
Description

Expose to start

The team whose members can start instances of this process in Process Portal. 

Expose performance metrics

The team whose members can view data for this process in the Process
Performance dashboard in Process Portal.
4. Click Save or Finish Editing.
