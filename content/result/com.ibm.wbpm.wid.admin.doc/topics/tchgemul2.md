# Disabling emulators

## About this task

## Procedure

1. In the test suite editor, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration
and module.
3 Complete one of the following steps:
    - If you want to disable a component or reference emulator,
expand the Component and Reference Emulators container.
    - If you want to disable a human task emulator, expand the Human
Task Emulators container.
4. Select the emulator that you want to disable.
5. In the Detailed Properties area,
select Disabled. The emulator will no longer
emulate the selected component, reference, or human task.
