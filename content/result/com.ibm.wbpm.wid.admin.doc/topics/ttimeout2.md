# Specifying timeout values for Wait On steps

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
Wait On step for which you want to specify a timeout value.
3. In the Timeout (ms) field of the
Detailed Properties area, select an input or output variable with
an integer data type. If the test data table does not contain any
input or output variable with an integer data type, you can create
one by selecting New. Information about creating
a new variable is found in the topic "Adding variables."
