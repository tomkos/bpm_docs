# Removing fine-grained traces

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration,
module, and Fine-Grained Traces containers.
3. In the Fine-Grained Traces container,
select one or more fine-grained traces that you want to remove.
4. Click the Remove icon . The selected fine-grained traces are removed
from the test configuration module.
