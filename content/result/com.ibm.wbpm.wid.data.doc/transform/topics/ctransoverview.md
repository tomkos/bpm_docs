# Functions and transforms

- IBM supplied transform types in the XML map editor

The XML map editor provides a ready-made set of transforms, each of which performs some action on the input data.
- XSLT and XPath functions

The XML map editor provides XSLT and XPath functions as primary transformations. XSLT 1.0 and XSLT 2.0, XPath 1.0 and XPath 2.0 functions are provided.
