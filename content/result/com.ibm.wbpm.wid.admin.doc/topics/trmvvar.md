# Removing test variations

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
test case for which you want to remove a test variation.
3. At the bottom of the test suite editor, click the Test
Data Table tab to open the Test Data Table view.
4. In the Test Data Table view, click the tab of the test
variation that you want to remove.
5. Click the Delete Variation icon . The selected test variation
is removed from the Test Data Table view.
