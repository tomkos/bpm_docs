Split Case pages

Split Case page

The Split Case page is used for creating new cases by reusing data from existing cases.

By default, when included on the Split Case page, the Case Toolbar widget includes the following buttons: Add and Cancel.

Split Case Adapter page

The default Split Case Adapter page is automatically added to all new case solutions. If you want, you can copy the default page and modify it in Page Designer to create your own custom page.
