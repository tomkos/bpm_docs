# Resolving issues with process instances by using the Gantt chart

- Adjust individual task due dates.
- Change the priorities of individual tasks.
- Adjust the due dates and priorities of several tasks at one time.
- Adjust the workflow for the tasks that are not yet started.

- Making multiple changes to process instance values in Process Portal

To resolve issues with an overdue process instance, you can change the settings of several tasks at one time. The Gantt chart can help you identify process instances that require attention.
- Auditing process data in Process Portal

In Process Portal, you can audit business data for process instances.
