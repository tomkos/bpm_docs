Earlier minor versions of Internet Explorer 10 or 11 can cause issues with workflow

applications

For example, you might find that in Case Builder, you cannot add a filter to an in-basket. In Case Client, you might find that some static pages do not open as expected.

To resolve these problems, use another supported browser to access your case management applications. If you want to use Internet Explorer 10 or 11, update the browser to the latest available version.
