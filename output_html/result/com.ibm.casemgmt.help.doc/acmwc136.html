Timeline Visualizer widget outgoing events

Select case event

| Description | The user selected the case in the Timeline Visualizer widget. | |---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------| | Event ID | icm.SelectTimelineCase | | Type | Broadcast | | Payload | timeline An icm.model.Timeline object that represents the Timeline API that is used to retrieve case history data. case An icm.model.Case object that represents the case that is associated with the timeline that the user selected timelineOverview An icm.model.TimelineOverview object that contains general case history data, such as the case identifier, name, and status; and events density data. |
