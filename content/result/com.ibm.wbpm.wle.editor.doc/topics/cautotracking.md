# Autotracking performance data for BPDs

- Custom KPIs associated with your BPD
- Custom KPIs associated with the activities in your BPD
- The Total Time KPI associated with each BPD by default

Autotracking is not required for the Process Performance and Team Performance dashboards, but
some features of the Process Performance dashboard are not available without autotracking. For more
information, see Enabling process instance management.

- Key performance indicators (KPIs) and service level agreements (SLAs)

With the data that IBM® Business Automation Workflow tracks and stores for key performance indicators (KPIs), you can analyze process performance and create service level agreements (SLAs).
- Creating custom KPIs

You can use key performance indicators (KPIs) to analyze process performance and create service level agreements. IBM Business Automation Workflow tracks KPI measurements at process run time and stores the information in the Business Data Warehouse.
- Associating KPIs with activities

To use a KPI in IBM Business Automation Workflow, you must associate it with one or several activities.
- Creating SLAs in Process Designer

 Traditional: 
 Create service level agreements (SLAs) so that you can analyze the performance of your business processes over time.
