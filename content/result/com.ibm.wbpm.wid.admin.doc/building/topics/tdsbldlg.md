# Disabling or enabling the Publish Changes window

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select Build Activities. The Build Activities
page opens.
3. Depending on whether you want to disable or enable the
View and Publish Changes to Servers window, either clear or select
the Show the window check box under the When
Update Running Servers is clicked and there are changes to publish
to the servers section.
4. Click OK.
