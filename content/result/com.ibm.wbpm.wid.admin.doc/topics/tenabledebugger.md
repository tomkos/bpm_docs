# Enabling or disabling the integration debugger

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select Debug. The Debug page opens.
3. Depending on whether you want to enable or disable the
integration debugger, either select or clear the Enable
business integration debugger (restart required) check
box.
4. Click OK.
5. Restart IBM Integration
Designer.
