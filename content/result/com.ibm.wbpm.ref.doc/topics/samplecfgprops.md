# Configuration properties for the BPMConfig command

The BPMConfig command
uses a properties file to configure your environment according to
the settings that you specify. Before you begin your configuration,
select a sample file that most closely resembles the configuration
that you want, copy the file, and customize it to suit your own environment.
This topic provides descriptions for the properties in the sample
properties files. It also provides descriptions for many properties
that are not contained in the sample properties files but that can
be manually added to the files.

To select a sample
properties file, see Sample configuration.properties files.
