# Enabling or disabling fine-grained trace

## About this task

## Procedure

1. Click Window > Preferences to open the Preferences window.
2. In the Preferences window, expand both Business
Integration and Integration Test Client and
then select Fine-Grained Trace. The Fine-Grained
Trace page opens.
3 Complete one of the following steps:
    - If you want to enable fine-grain trace in the integration
test client, ensure that the Enable fine-grained trace
support when starting the test client check box is selected.
    - If you want to disable fine-grain trace in the integration
test client, clear the Enable fine-grained trace support
when starting the test client check box.
4. Click OK.
