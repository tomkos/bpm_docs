# Setting up autotracking

## About this task

## Procedure

1. Open the process in Process Designer.
2. Click the Tracking tab for the process.
3. Ensure Enable Autotracking is selected.
4. In the Autotracking Name field,
enter the name you want to use.
5 In order to analyze process data according to particularbusiness variable values, set the variables as follows.
    1. Click the Variables tab for the diagram.
    2. For each variable you want to track, right click it
and then click Track this Variable.
6. Click Save or Finish Editing.
7. To send the new tracking requirements to the Business Performance
Data Warehouse, in the Overview page of the Process
App Settings, click Update Tracking Definitions.
