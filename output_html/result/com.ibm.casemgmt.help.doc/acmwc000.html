Attachments widget events

  • Attachments widget outgoing events

The Attachments widget provides the following outgoing events that are broadcast and wired to the other widgets on the page. The data that is included in the payload of a broadcast event is processed by any widget that has a corresponding handled event. The data that is included in the payload of a wired event is processed by the handled event in the target widget to which the published widget is wired. - Attachments widget incoming events

The Attachments widget provides incoming events to handle the data that is received from other widgets.
