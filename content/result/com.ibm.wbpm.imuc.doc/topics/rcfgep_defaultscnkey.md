# Default Business Automation Workflow endpoint
scenario keys

| Default IBM BPM endpoint scenario keys   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| EXTERNAL\_CLIENT                          | This generic scenario key is intended for non-relative URLs to be used by clients that are outside the data center, such as web browsers, to connect to the Business Automation Workflow server.Â  If all external clients connect directly to the Business Automation Workflow server, there is no need to change the default value. If all external clients use one entry point into your network, such as a web server, you can define the EXTERNAL\_CLIENT scenario to use a static virtual host object or a static URL. If you have multiple entry points for external clients, for example, a load balancer and a web server, using a single virtual host setting or a static URL is not appropriate, and you must define the EXTERNAL\_CLIENT scenario to use one or more dynamic strategies. Use the Strategies for identifying endpoint information topic to decide which strategies you will use. |
| INTERNAL\_CLIENT                          | This generic scenario key is intended for non-relative URLs to be used by clients that are inside the data center, such Business Automation Workflow calling itself.  If all internal clients connect directly to the Business Automation Workflow server, there is no need to change the default value. If all internal clients use one entry point into your network, such as a web server, you can define the INTERNAL\_CLIENT scenario to use a static virtual host object or a static URL. If you have multiple entry points for internal clients, for example, a load balancer and a web server, using a single virtual host setting or a static URL is not appropriate, and you must define the INTERNAL\_CLIENT scenario to use one or more dynamic strategies. Use the Strategies for identifying endpoint information topic to decide which strategies to use.                                   |
| RELATIVE                                 | This generic scenario key is intended for relative URLs to facilitate access to browser-based web applications through various entry points in your topology. For example, some users might access Business Automation Workflow through a reverse proxy whereas others might connect directly to the web server.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
