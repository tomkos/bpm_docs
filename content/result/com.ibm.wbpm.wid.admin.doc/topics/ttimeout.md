# Specifying the timeout value for the test client

## About this task

## Procedure

1. Click Window > Preferences to open the Preferences window.
2. In the Preferences window, expand Business Integration and
select Integration Test Client. The Integration
Test Client page opens.
3. In the Test client timeout (in minutes) field,
select the number of minutes that you want to use for the test client
timeout value.
4. Click OK.
