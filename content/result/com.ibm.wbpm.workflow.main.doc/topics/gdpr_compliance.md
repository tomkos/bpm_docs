# GDPR readiness

- Configuring Business Automation Workflow for GDPR readiness

Certain configuration options and settings are required for GDPR readiness.
- Developing GDPR-ready process applications

When developing applications special consideration needs to be given to personal data, where it is stored, how it is retrieved, where it is used, and how it can be deleted or anonymized. You must also secure and restrict access to applications that handle sensitive data.
- Developing GDPR-ready solutions and cases

Choices about how solutions handle the property values that store personal data affect how easy that data is to retrieve or delete.
