# Enabling or disabling debugger tracing on servers

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select Debug. The Debug page opens.
3. Depending on whether you want to enable or disable integration
debugger tracing on the server, either select or clear the Enable
debugger tracing on the server (restart required) check
box.
4. Restart your server or restart IBMÂ® Integration
Designer.
5. Click OK.
