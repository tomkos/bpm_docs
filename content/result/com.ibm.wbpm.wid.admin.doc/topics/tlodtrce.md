# Loading test traces for the integration test client

## About this task

## Procedure

1. In the Business Integration view, right-click a module
and select Test > Load test trace. The Load
Test Trace wizard opens.
2. In the Select a file list box, navigate
to your test trace file and select it. (By default, test trace files
have a .wbiexetrace file extension.)
3. Click Finish. The test trace opens
in a new instance of the integration test client.
