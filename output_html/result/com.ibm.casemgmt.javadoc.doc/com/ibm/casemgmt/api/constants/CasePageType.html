Enum CasePageType

public enum CasePageType extends java.lang.Enum Represents the available case page types. When a solution is deployed, a particular mashup page can be assigned to various page types for each case type. For the CASE_PAGE type, the particular page can be different for each role.

Each value in the enumeration is associated with a string representation of that value.
