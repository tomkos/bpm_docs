# Adding test cases to a test bucket configuration

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. Click the Add icon . The New Configuration wizard opens.
3. In the Wizards list box, select Test
Case and click Next. The Select
a Test Bucket Configuration page opens.
4. In the Select test bucket configuration to continue list,
select the test bucket configuration to which you want to add one
or more test cases.
5. Click Next. The Add Test Cases page
opens.
6. In the Select the test suites and test cases
to add to the test bucket configuration list, ensure that
the check box is selected beside the name of each component test project,
test suite, or test case that you want to add to the test bucket configuration.
7. Click Finish. The selected component
test projects, test suites, and test cases are added to the test bucket
configuration.
