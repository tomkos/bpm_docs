# Adding modules to a test configuration

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. Click the Add icon . The New Configuration wizard opens.
3. In the Wizards list box, select Module and
click Next. The Select a Test Configuration
page opens.
4. In the Select a test configuration to continue list
box, select the test configuration to which you want to add one or
more modules.
5. Click Next. The Add Modules page
opens.
6. In the Select modules in this configuration list
box, ensure that the check box is selected beside the name of each
module that you want to add to the test configuration.
7. Click Finish. The modules are added
to the selected test configuration.
