Enum CommentType

  • All Implemented Interfaces: java.io.Serializable, java.lang.Comparable

public enum CommentType extends java.lang.Enum Represents a type of comment applied to a case. Comments are always attached to a case, but depending on their type may also apply to another object type related to that case.

Each value in the enum is associated with a string representation of that value.
