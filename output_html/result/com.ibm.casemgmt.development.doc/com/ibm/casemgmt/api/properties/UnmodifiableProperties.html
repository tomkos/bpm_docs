Interface UnmodifiableProperties

  • public interface UnmodifiableProperties Represents a non-modifiable interface to a collection of properties. The UnmodifiableProperty interfaces accessed through this collection allow the values and other attributes of properties to be accessed but the values cannot be modified.

    This interface can be obtained from a Case Management Java API object, such as a Case object, when there is no intent to modify the properties of the object. In that scenario some communication to the server can be optimizedIn particular, communication with an external data service can be deferred until there is an intent to modify the object.

    Most of the methods on this interface correspond to non-modifying methods on the CaseMgmtProperties class.
