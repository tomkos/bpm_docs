# Creating static relationships with business objects

## About this task

## Procedure

1. Start IBMÂ® Integration
Designer.
2. Click File > New > Relationship.
3. Provide a Name for the relationship and click Next.
4. Select the "Used to transform data attributes according
to a static mapping" check box for static relationships.
5. Click Finish.
6. You can then add roles to the relationship by choosing
business objects from the list.
7. Next, add keys to each role business object.
