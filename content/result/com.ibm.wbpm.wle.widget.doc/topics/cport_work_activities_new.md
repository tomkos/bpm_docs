# Working on activities in Process Portal

If a process instance has activities, they are shown in the Activities section
of the instance details page. The section is empty if the instance
has no activities.

| Filter      | Status of activities                          | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-------------|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ready       | Ready or waiting                              | The activities that are ready to be started manually and activities that are waiting for start conditions, such as the filing of a document or the updating of business data, to be fulfilled so that they can start.  Actions you can take on ready activities  Start the activity (click Start). Claim and start the activity (click Start >  Assign to Me and Start). You become the owner of the task that is associated with the activity. The task details page opens for you to complete the work that is required on the task. Reassign the activity to someone else or to the owner team (click Start >  Reassign and Start). The task that is associated with the activity is transferred to the new owner.  For all actions, the started activity is transferred to the In Progress list. Actions you can take on waiting activities  None |
| In Progress | Working or waiting                            | The activities that are started and activities that are waiting to be started automatically. Actions that you can take None                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Completed   | Completed or failed                           | The activities that are in an end state.  Actions you can take None                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| All         | Ready, waiting, working, completed, or failed | This list shows all the activities in all states for the instance. You can perform actions only on activities that are marked as ready.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
