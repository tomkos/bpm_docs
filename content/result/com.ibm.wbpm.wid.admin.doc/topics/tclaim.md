# Specifying a wait time for human task claims

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration,
module, and Human Task Emulators container.
3. Select the human task emulator for which you want to specify
the wait time.
4 In the Detailed Properties area,choose one of the following options:
    - If you want to have the human task claimed immediately, select Immediately.
    - If you want to specify an amount of time to wait before the
human task is claimed, select After (ms) and
then type the number of seconds that you want to wait in the After
(ms) field.
