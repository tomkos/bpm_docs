# Adding fine-grained traces

## About this task

## Procedure

1. In the test suite editor, click the Configurations tab.
The Configurations page opens.
2. Click the Add icon . The New Configuration wizard opens.
3. In the Wizards list, select Fine-Grained
Trace and click Next. The Select
a Module page opens.
4. In the Select a module to continue list,
locate the test configuration that contains the module where you want
to add a fine-grained trace, then select the module.
5. Click Next. The Select Components
to Trace page opens.
6. In the Choose the components list,
ensure that the check box is selected beside the name of each component
for which you want to add a fine-grained trace.
7. Click Finish. The fine-grained traces
are added to the selected test configuration module and appear under
the Fine-Grained Traces container.
