# Changing Java breakpoint query timeout values

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the left frame of the Preferences window, expand Business
Integration and select Debug. The
Debug page opens.
3. In the Java breakpoint query timeout (in milliseconds) field,
specify a value in milliseconds. The default value is 180000 milliseconds,
which is 3 minutes.
4. Click OK.
