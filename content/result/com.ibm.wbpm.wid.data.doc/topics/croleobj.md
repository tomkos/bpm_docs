# Role objects

When a role object is selected, the general properties page will
open. From here you have the information about the referenced Name, Display
name, Namespace and Folder in
the file system. The Display name can be edited and the changes will
be reflected in the editor.

- edit the existing one using the business object editor
- select a different data type as new role object
- drag a data type from the Business Integration View and drop it
onto the role or role object
