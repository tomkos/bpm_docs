## About this task

## Procedure

1. Open a command window and change to the directory where
the ws\_ant.bat file resides. The default directory
is BPM\_home/bin (where BPM\_home  is the installation
path of IBMÂ® Business Automation
Workflow in IBM Integration
Designer).
For example, C:\Program Files\IBM\WID75\_WTE\runtimes\bi\_v75\bin.
2. Run the command ws\_ant.bat -f build\_file\_path (where build\_file\_path is
the absolute path to the build file). For example: ws\_ant.bat
-f C:\MyScripts\build.xml.
