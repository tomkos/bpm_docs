# Adding Wait for Time steps

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
test case invocation for which you want to add a Wait for Time step.
3. Click the down arrow beside the Add Test Case icon .
4. From the list, select Add Wait For Time.
A new Wait for Time step is displayed under the associated invocation
in the Test Cases area.
