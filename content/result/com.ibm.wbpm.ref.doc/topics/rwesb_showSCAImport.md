# showSCAImport command

## Location

Start the wsadmin scripting client from the install\_root/profiles/deployment\_manager\_profile/bin directory.

## Syntax

```
showSCAImport
-moduleName moduleName
-import import\_name
[-applicationName applicationName]
```
