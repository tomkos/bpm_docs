# manageprofiles parameters

The manageprofiles command-line utility is in the
						install\_root/bin directory, where
					install\_root is the installation location of IBM Business
				Process Manager. The command file is a script named
					manageprofiles.sh for LinuxÂ® and UNIX platforms, or
					manageprofiles.bat for Windows platforms. For more information, see manageprofiles command-line utility.
