# Adding Wait On steps

## Before you begin

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
asynchronous invocation for which you want to add a Wait On step.
3. Click the down arrow beside the Add Test Case icon .
4. From the list, select Add Wait On.
A new Wait On step is displayed under the associated asynchronous
invocation in the Test Cases area.
