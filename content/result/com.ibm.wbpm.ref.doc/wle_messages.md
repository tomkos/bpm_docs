# Workflow Center and Process Designer messages

```
AAAANNNNT
```

- AAAA represents the component identifier
- NNNN represents a four digit identifier
- T represents the type of messages. There are three message types:
    - I represents informational messages
    - W represents warning messages
    - E represents error messages

The following component identifiers are used for Workflow Center and Process Designer messages.

- CWLLG
- CWLND
- CWMCO
- CWMHW
- CWPFD
- CWTBG
- CWTBI
- CWTCA
- CWTDE
- CWTDM
- CWTDS
- CWXSD
