# Redefining emulators as manual

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand
the test configuration and module containers.
3 Complete one of the following steps:
    - If you want to redefine a component or reference emulator
as manual, expand the Component and Reference Emulators container.
    - If you want to redefine a human task emulator as manual, expand
the Human Task Emulators container.
4. Select the emulator that you want to redefine as manual.
5. In the Detailed Properties area,
select Manual emulation. The emulator is now
redefined as a manual emulator and you can immediately use it for
testing.
