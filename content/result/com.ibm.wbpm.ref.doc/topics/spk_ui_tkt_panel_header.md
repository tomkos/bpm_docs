# Panel header

## Example

As an example, the Panel header view can contain a nested Date Picker view (deprecated), and as a
result date picker is displayed as the header of the related panel section.

## Events

- On load: Activated when the page loads.

Depending
on the specific event, you can use JavaScript logic to modify the effects of the
view.
