# Launching the Business Process Rules Manager

## Before you begin

Make sure that both the server and
client are configured correctly.

- https://hostname:9443/br (if security is
enabled)
- http://hostname:9080/br (if security is
not enabled)

## About this task

## Procedure

1. Switch to the server view.
2. Start your server.
3. Right-click the server and select Launch > Business Process Rules Manager.
4. At the Login page, type your User ID.
5. Type your Password
6. Click Login. The
business process rules manager Initial page opens.
