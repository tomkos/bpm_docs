# Specifying the maximum number of previous input values to save

## About this task

## Procedure

1. Click Window > Preferences to open the Preferences window.
2. In the Preferences window, expand Business Integration and
select Integration Test Client. The Integration
Test Client page opens.
3. In the Maximum number of previous input values
to save field, select the maximum number of previous input
values that you want to save when you close the test client.
4. Click OK.
