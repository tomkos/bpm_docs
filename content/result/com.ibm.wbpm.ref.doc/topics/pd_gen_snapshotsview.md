# SNAPSHOTS view

The SNAPSHOTS view includes the following columns:

| Column      | Description                                                                                |
|-------------|--------------------------------------------------------------------------------------------|
| SNAPSHOT\_ID | The primary key of this snapshot in the Business Performance Data Warehouse.               |
| NAME        | The name assigned to this snapshot in the Workflow Center.                                 |
| DESCRIPTION | The description given to this snapshot, if one was provided, in the Workflow Center.       |
| ACRONYM     | The acronym designated for the process application or toolkit that contains this snapshot. |
