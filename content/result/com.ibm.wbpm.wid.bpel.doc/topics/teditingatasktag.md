# Editing task tags

## About this task

## Procedure

1. In the Javaâ„¢ perspective,
select Window > Preferences. The Preferences window
opens.
2. Expand Java > Compiler, and then
click Task Tags.  The Task
Tags window opens.
3. Highlight the task tag you want to edit, and then click Edit.
The Edit Task Tag window opens.
4. In the Edit Task Tag window, specify
a different name or priority for the task tag, as required.
5. Click OK. The Task
Tags window shows the edited task tag.
