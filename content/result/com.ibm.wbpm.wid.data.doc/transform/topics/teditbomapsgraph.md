# Editing business object maps using the graph view

## About this task

## Procedure

1. Click Show graph view on the toolbar to
switch to the graph view.
2. The canvas shows the business object map and its elements in the
graph view.
3. From the canvas in the graph view, you can select the business
object of the business object map to change as needed. The elements can be
altered or deleted. For example, to edit a business object in the map, select
the business object in the graph view and right click and select Edit
Business Object. You can also
edit a business object by double-clicking the business object name or by pressing F3.
