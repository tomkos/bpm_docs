Adding workflow processes to existing solutions

Procedure

To add the workflow actions to an existing solution:

  1. Open the solution in Case Builder.
  2. On the Pages tab, edit all solution pages that contain the In-baskets widget. By default, the In-baskets widget is included on the static Work page.
  3. In Page Designer, click the Edit Settings icon of the In-baskets widget. 4 On the Menus tab, add the following actions to the menu:
    • Modify Task
    • View Process
    • Modify Process
    • View Process Diagram
    • Audit History
  4. After you update the relevant pages, save and redeploy the solution.
