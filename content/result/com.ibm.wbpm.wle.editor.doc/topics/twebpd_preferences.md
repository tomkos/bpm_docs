# Setting preferences in the Process Designer

## About this task

- Choose the locale for your Process Designer
that defines the language settings and formatting conventions for other elements such as date and
time, currency, and number format.
- Hide or show a warning about possible errors when the process details UI or launch UI interface
changes, and you choose not to regenerate the rest of the client-side human service components.
- Hide or show a warning that the tracking definition for a variable will be deleted when the
variable definition changes.
- Define the default values for the server URL and email address for Blueworks Live process subscriptions.
- Show or hide the
runtime performance monitor that displays coach performance statistics.

## Procedure

1. In the upper-right corner of the Process Designer, click your user name, and select
Preferences.
2. Make the changes to your settings, and click Save.
