public final class SimpleP8ConnectionCache extends BaseP8ConnectionCache A basic implementation of P8ConnectionCache that is appropriate to cache the Connection for a single user in the context of a thread of execution. It caches Connection objects in a map of this instance so it is dedicated to the current calling thread.
