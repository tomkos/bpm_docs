# Using external implementations in BPDs (deprecated)

To use an external implementation to implement an activity in a
business process definition (BPD), complete the tasks listed in this
section.

1. Building a custom application to implement an activity (deprecated)

 Traditional: 
Build a custom application using the IBM Business Automation Workflow Web API to implement an activity in a BPD.
2. Creating an external implementation (deprecated)

 Traditional: 
Create an external implementation when you want to reuse an existing external application or create an external application to handle one or more steps in your process.
3. Creating an external implementation to implement an activity (deprecated)

 Traditional: 
Select a custom application to implement a particular activity (step) in a BPD.
