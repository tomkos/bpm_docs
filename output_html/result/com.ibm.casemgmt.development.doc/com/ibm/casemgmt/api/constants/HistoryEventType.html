Enum HistoryEventType

  • java.lang.Object

    • java.lang.Enum
      • com.ibm.casemgmt.api.constants.HistoryEventType

  • All Implemented Interfaces: java.io.Serializable, java.lang.Comparable

public enum HistoryEventType extends java.lang.Enum Represents a type of case history event. Each event type is associated with a string value which is the symbolic name of a core Content Engine event class or an ICM custom event class.

ID status: ID review by David Newhall, 5/20/2012.
