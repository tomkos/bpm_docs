# Panel footer

## Example

As an example, the Panel footer view can contain a nested Image view, and as a result this image
is displayed as the footer of the related panel section.

## Events

- On load: Activated when the page loads.

Depending
on the specific event, you can use JavaScript logic to modify the effects of the view.
