# Difference between heritage human services and client-side human services

| Compare                           | Heritage human services (deprecated)                                                                                                                                                                                                                                                                                                                                                                                            | Client-side human services                                                                                                                                                                                                                                                                                                                                             |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Authoring in IBM Process Designer | Desktop Process Designer or Process Designer                                                                                                                                                                                                                                                                                                                                                                                    | Process Designer                                                                                                                                                                                                                                                                                                                                                       |
| Execution environment             | On the server                                                                                                                                                                                                                                                                                                                                                                                                                   | Client-side, in the web browser, with calls for data to the server only when necessary.                                                                                                                                                                                                                                                                                |
| Collaboration (deprecated)        | Supported                                                                                                                                                                                                                                                                                                                                                                                                                       | Not supported                                                                                                                                                                                                                                                                                                                                                          |
| Page support                      | Both pages and heritage coaches (deprecated) are supported in the desktop Process Designer and the Process Designer. Creation of heritage coaches is not supported in the Process Designer. Basic support is provided to open and edit the XML model of heritage coaches in the Process Designer.                                                                                                                               | Pages are supported in client-side human services, however heritage coaches are not.                                                                                                                                                                                                                                                                                   |
| View support                      | Full support for coach views in the deprecated Coaches toolkit, which can be used on desktop devices only. Support for views in the UI toolkit and the deprecated Responsive Coaches toolkit, without the ability to use or set the responsive behavior of the views in the desktop Process Designer. Responsiveness is supported in the Process Designer. Deprecated coach views in the Coaches toolkit do not support themes. | Full support for coach views in the deprecated Coaches toolkit, which can be used on desktop devices only. Full support for views in the UI toolkit and the deprecated Responsive Coaches toolkit. The responsive views can be used on multiple device types, such as mobile and desktop devices. Deprecated coach views in the Coaches toolkit do not support themes. |
| Theme support                     | Available The predefined Carbon theme is the default theme for all new and migrated process applications Custom theme definitions can be edited in the theme editor                                                                                                                                                                                                                                                             | Available The predefined Carbon theme is the default theme for all new and migrated process applications Custom theme definitions can be edited in the theme editor                                                                                                                                                                                                    |
| Usage of JavaScript               | Server scripts that use server script. syntax                                                                                                                                                                                                                                                                                                                                                                                   | Client-side scripts that use standard JavaScript syntax.                                                                                                                                                                                                                                                                                                               |
| Nested service support            | Available for nested heritage human services.                                                                                                                                                                                                                                                                                                                                                                                   | Available for nested client-side human services. Multiple nested layers are supported.                                                                                                                                                                                                                                                                                 |
| Number of outgoing connections    | Multiple outgoing connections from nested services.                                                                                                                                                                                                                                                                                                                                                                             | A single outgoing connection from a nested service.                                                                                                                                                                                                                                                                                                                    |
