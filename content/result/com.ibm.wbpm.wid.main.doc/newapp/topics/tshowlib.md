# Hiding or displaying libraries and other projects

If you are working with an integration solution diagram
in the integration solution editor and your diagram is consists of
many modules or mediation modules, you can improve the visibility
of the diagram by hiding associated projects like libraries, Java
projects, component test projects, and simple projects.

## About this task

## Procedure

1. If you want to display libraries and other projects, click
the Show Libraries and Other Projects icon . The libraries
and other projects are displayed and the icon toggles to the Hide
Libraries and Other Projects state.
2. If you want to hide libraries and other projects, click
the Hide Libraries and Other Projects icon . The libraries
and other projects are hidden from view.
