# Unsupported  EXSLT functions

The following EXSLT functions are not supported at run time and
are no longer available in the XML map editor.

- date:add()
- date:add-duration()
- date:difference()
- date:duration()
- date:parse-date()
- date:seconds()
- date:sum()
- date:week-in-month()
- dyn:closure() I
- dyn:evaluate() I
- dyn:map() I
- dyn:max() I
- dyn:min() I
- dyn:sum() I
- exsl:document()
- random:random-sequence()
- regexp:match()
- regexp:replace()
- regexp:test()
- str:decode-uri()
- str:encode-uri()
- str:replace()
