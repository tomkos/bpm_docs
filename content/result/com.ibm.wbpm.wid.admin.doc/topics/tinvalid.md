# Specifying how invalid breakpoints should be handled in the
integration debugger

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select debug. The debug page opens.
3 In the Cleanup invalid breakpoints radiobutton group, choose one of the following radio buttons:
    - Select Always to have invalid breakpoints
automatically removed. This is the default selection.
    - Select Never to prevent invalid breakpoints
from being automatically removed.
    - Select Prompt to be prompted for a
decision on whether to automatically remove invalid breakpoints whenever
they are detected.
4. Click OK.
