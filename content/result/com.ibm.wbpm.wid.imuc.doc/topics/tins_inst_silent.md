# Installing IBM Integration
Designer silently

You can install the IBM Integration
Designer
product package in silent installation mode. When you install in silent mode, the user
interface is not available.

## About this task

- Installing IBM Integration Designer silently using the command line

You can install IBM Integration Designer using the command line.
- Installing IBM Integration Designer silently using a response file

You can install the IBM Integration Designer product package in silent installation mode. When you install in silent mode, the user interface is not available. Instead, you run a command to use a response file that installs the product.
- Creating and configuring the test environment profile using BPMConfig

You can use the BPMConfig command to create and configure the profile for the IBM Workflow Server test environment using a properties file that contains all of the values required for the profile settings.
