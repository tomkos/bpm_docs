GET method for the list of solutions resource

URI

/CASEREST/v1/solutions

Request content

The request for this method contains no JSON content.

Response content

  • Solution name
  • Name of the target object store to which the solution is deployed
  • Deployment status
  • Connection point
  • Web application ID

| Code | Description | |---------------------------|-------------------------------------------------------------------------------------------------------------| | 200 OK | The method completed successfully. The requested list of solutions was returned. | | 404 Not Found | No solutions were found.For information about the error, see the userMessage element in the JSON response. | | 500 Internal Server Error | A server error occurred. For information about the error, see the userMessage element in the JSON response. |
