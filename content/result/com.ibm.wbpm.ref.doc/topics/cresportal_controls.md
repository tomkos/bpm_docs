# Coach views in the Responsive Portal Components toolkit

- An HTML file that contains the AngularJS template that defines
the user interface.
- A JavaScript controller
file containing the AngularJS controller that controls the behavioral
logic of the coach view.
- A JavaScript preview
file that shows a preliminary view of the coach view when it is dropped
on the coach editor palette.

| Coach view            | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|-----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Action Bar            | Provides an action bar for the user interface. The coach view includes a center title, extended bar details, and two flanking icon buttons. The icon buttons can have a numerical overlay. To ensure that the coach view is accessible for keyboard navigation and screen readers, configure it so that it has focus when the Process Portal page loads.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service for the action bar on the dashboard pages. All the buttons are enabled. The left button hides and displays the main menu, the center title hides and displays actions that are specific to the page, and the right button hides and displays the context menu. |
| Coach Viewer          | Displays either coaches in an iFrame or the Work dashboard. You can use the coach view to complete a task or to view a dashboard.In Process Portal, Coach Viewer is used in the RESPONSIVE\_PORTAL client-side human service to display either the coach views for dashboard pages, or to display the Work dashboard, which is also used for displaying saved searches.                                                                                                                                                                                                                                                                                                                                               |
| Coach Viewer Modal    | Displays coaches in a modal dialog. Coach Viewer Modal presents coach views only if it is not running in an iFrame. If the coach view is running in an iFrame, it relays the coach-viewing event to the parent document. You can use the coach view to complete a task or to view a dashboard in a modal dialog.In Process Portal, Coach Viewer Modal is used in the RESPONSIVE\_PORTAL client-side human service to display the coach views for dashboard pages.                                                                                                                                                                                                                                                     |
| Dashboard Context     | Shows the posts in which the logged-on user is mentioned, the process instances that the user is following, and exposed links if they are available.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service and is shown in the context menu for the dashboard page.                                                                                                                                                                                                                                                                                                                                                                                                           |
| Launcher              | Provides a list of the dashboards, saved searches, launchable processes, and services that are exposed to the current user.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service to show the dashboards, saved searches, launchable processes, and services in the main menu.                                                                                                                                                                                                                                                                                                                                                                                                |
| Main Pane             | Provides the main pane for the user interface. The coach view contains a content box. To design the user interface, drag other coach views onto the content box. To ensure that the coach view is accessible for keyboard navigation and screen readers, configure a focus keyboard shortcut that can be used with Ctrl+Alt.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service as a container for the coaches that comprise the dashboard page.                                                                                                                                                                                                                           |
| Modify Instance Modal | Provides a modal dialog for updating a process instance. Users must interact with the dialog before they can return to the main window.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service to display the modal dialog where authorized users can change instance details.                                                                                                                                                                                                                                                                                                                                                                                                 |
| Modify Task Modal     | Provides a modal dialog for updating a task. Users must interact with the dialog before they can return to the main window. In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service to display the modal dialog where authorized users can change task details.                                                                                                                                                                                                                                                                                                                                                                                                                |
| Notifications         | Provides a toast notification to users when an event occurs that affects their work.In Process Portal, this coach view is used in both the RESPONSIVE\_PORTAL and RESPONSIVE\_WORK client-side human services.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Saved Search Editor   | Provides a coach view for creating saved searches.In Process Portal, this coach view is part of the Task Filter coach view. Both coach views are used in the RESPONSIVE\_WORK client-side human service.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Side Pane             | Provides a side pane in the user interface. To ensure that the coach view is accessible for keyboard navigation and screen readers, configure a focus keyboard shortcut that can be used with Ctrl+Alt.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service to provide the containers for the main menu and context menu.                                                                                                                                                                                                                                                                                                                                                   |
| Task Context          | Provides a container for context information that is associated with the task. In Process Portal, the coach view is implemented as a composite coach view in the RESPONSIVE\_PORTAL client-side human service and is shown in the context menu for the task completion. The coach view contains the Task Details, Task Experts, and Task Stream coach views. These coach views provide the corresponding tabs in the context menu for task completion.                                                                                                                                                                                                                                                                |
| Task Details          | Provides a coach view that shows information about the process instance that is associated with a task. It includes the business data, links to the process instance details, and all the tasks in the process. Users can also follow or unfollow the instance from this coach view.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service as part of the composite Task Context coach view. The coach view provides the Details tab in the context menu for task completion.                                                                                                                                                                                                 |
| Task Experts          | Provides a coach view that shows the experts for the current task. Depending on how you configure the coach view, it can include both the subject matter experts from the Experts team, and the experienced users for the task that are identified by the system.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service as part of the composite Task Context coach view. The coach view provides the Experts tab in the context menu for task completion.                                                                                                                                                                                                                    |
| Task Filter           | Provides a coach view that allows users to filter the task list by entering text. In Process Portal, this coach view is used in the RESPONSIVE\_WORK client-side human service to provide the search field at the top of the task list in the Work page. It also contains the Saved Search Editor coach view.                                                                                                                                                                                                                                                                                                                                                                                                         |
| Task List             | Provides a coach view that displays tasks as a list or a table. In Process Portal, this coach view is used in the RESPONSIVE\_WORK client-side human service to provide the task list in the Work page.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Task Stream           | Provides a coach view that shows the aggregated postings, comments, system events, and completed tasks for the process instance that is associated with a task. Users can post comments to the stream. Depending on how you configure the coach view, users can also post files and links to the stream.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service as part of the composite Task Context coach view. The coach view provides the content in the Stream tab in the context menu for task completion.                                                                                                                                                               |
| User Profile          | Provides a coach view with user name, edit profile link, and log out link. The edit profile link opens a modal dialog for editing user profiles, which allows you to configure avatar, contact information, language settings, calendar preferences, notification behavior, and other preferences.In Process Portal, this coach view is used in the RESPONSIVE\_PORTAL client-side human service and it is displayed at the top of the main menu.                                                                                                                                                                                                                                                                     |
