# Removing events in the integration test client

## About this task

## Procedure

1. In the integration test client, click the Events tab.
The Events page opens.
2 In the Events area, complete one of the following steps:
    - If you want to remove one or more Invoke or Attach events,
select the Invoke and Attach events that you want to remove and then
right-click one of them and select Remove.
The selected events and any associated events are removed from the
Events area.
    - If you want to remove all events, right-click any event and
then select Remove All. All events are removed
from the Events area.
