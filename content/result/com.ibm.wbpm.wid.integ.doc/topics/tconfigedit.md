# Editing a binding resource configuration

## Before you begin

## About this task

## Procedure

1. Expand Binding Resources in the
Business Integration navigation.
2. Double-click the configuration you want to edit.
3. Edit the Name and Namespace fields, if you
want to edit them.
4. Use the Browse button to select
a different class name for the Class name field.
5. In the Select bindings field, change or add different
bindings by selecting them.

## What to do next

You can configure your binding in the Properties view
by selecting a data handler, if one or more have been created, or
creating a new data handler.
