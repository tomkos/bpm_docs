# Editing fine-grained traces

## About this task

## Procedure

1. In the test suite editor, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration,
module, and Fine-Grained Traces containers.
3. In the Fine-Grained Traces container,
select the fine-grained trace that you want to edit.
4. In the Detailed Properties area,
select the variables that you want to track and display in the value
editor. You can select business process variables or state machine
variables, but mediation flow messages are automatically collected
and cannot be individually selected.
