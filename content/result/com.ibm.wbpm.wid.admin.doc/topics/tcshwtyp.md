# Showing variable or message type names in the integration debugger

## About this task

## Procedure

1. In the Debug view, ensure that the correct stack frame
is selected for the variables or messages whose type names you want
to show or hide.
2. In the Variables view, click the arrow icon .
3. From the menu, select Layout  and
remove the check mark beside the Show Columns menu
item.
4. In the Variables view, click the Show Type Names icon . The icon toggles between showing and
hiding the type names of the variables or messages.
