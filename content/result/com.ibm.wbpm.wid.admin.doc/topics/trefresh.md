# Refreshing invocations

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, locate the
invocation that references the operation that you want to refresh.
The invocation will be flagged with an error symbol.
3. Right-click the invocation and select Refresh.
The error symbol disappears from the invocation.
