# isDefaultWXSDefinition command

```
$AdminTask help WXSAdminCommands
```

The isDefaultWXSDefinition command
is run using the AdminTask object of the wsadmin scripting client.

## Location

The wsadmin scripting client is
located in the install\_root\bin directory.

## Syntax

```
isDefaultWXSDefinition
-name definitionName
```

## Required parameter

## Example

```
AdminTask.isDefaultWXSDefinition('[-name MydefName]')
```
