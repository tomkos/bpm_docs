# Opening parents from within child event definitions in the
event definition editor

If you are editing an event definition, you can easily
open the parent of the event definition from within the event definition
editor. However, you cannot open the parent event definition if it
is the WBI.MonitoringEvent root event definition.

## About this task

## Procedure

1. In the event definition editor, insert your cursor in either
the Name field or the Parent field.
2. Right-click and select Open Parent Event Definition.
The parent event definition opens in another event definition editor
window.
