Class ExternalDocumentReference

  • java.lang.Object

    • com.ibm.casemgmt.api.ExternalDocumentReference

  • public final class ExternalDocumentReference extends java.lang.Object Represents a reference to an external document filed in a Case folder or a sub-folder of a Case. References to external documents that are filed in the same Case folder or sub-folder share a single proxy document which is multi-filed into each folder. This class takes care of locating the existing proxy document, if any, and filing that same document into a different folder, or creating a new proxy document if none exists.
