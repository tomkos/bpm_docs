# Adding test configurations to the integration test client

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. Click the Add icon . The New Configuration wizard opens.
3. In the Wizards list box, select Test
Configuration and click Next. The
Create a Test Configuration page opens.
4. In the Test configuration name field,
type the name that you want to assign to the new test configuration.
5. In the Selected modules in this configuration list
box, select the check box beside the name of each module that you
want to add to the new test configuration.
6. Click Finish. The test configuration
is added to the Configurations area.
