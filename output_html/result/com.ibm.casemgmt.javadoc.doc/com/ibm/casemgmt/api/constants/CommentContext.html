Enum CommentContext

  • java.lang.Object

    • java.lang.Enum
      • com.ibm.casemgmt.api.constants.CommentContext

  • All Implemented Interfaces: java.io.Serializable, java.lang.Comparable

public enum CommentContext extends java.lang.Enum Represents the action, such as adding a document or case, that was being taken when a comment was created.

Each value in the enum is associated with an integer value from the Content Engine choice list CmAcmActionChoiceList. It is also associated with a string representation of that value.
