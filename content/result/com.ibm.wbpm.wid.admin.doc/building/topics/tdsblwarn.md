# Controlling prompts about the Build Automatically menu item

## About this task

## Procedure

1. From the Window menu, select Preferences.
The Preferences window opens.
2. In the Preferences window, expand Business Integration and
select Build Activities. The Build Activities
page opens.
3. Under the When the Build Automatically menu
item is not in the selected state and a build activity setting is
changed in the Build Activities view section, either clear
or select the Show the window check box.
4. Click OK.
