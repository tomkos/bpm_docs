# Integer (deprecated)

<!-- image -->

## Restrictions and limitations

## Data binding

| Binding description                                                                                                                             | Data type   |
|-------------------------------------------------------------------------------------------------------------------------------------------------|-------------|
| Contains the number that this control displays. The Integer can be empty and the Integer control saves the number that the user enters into it. | Integer     |

## Configuration properties

| Configuration property    | Description                                                                                                                                                                                                          | Data type   |
|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------|
| Hide Thousands Separators | Select this option to hide the delimiter characters used to separate thousands. For example, select this option to display ten thousand as 10000 instead of 10 000 or 10,000. The default is to show the separators. | Boolean     |
| Minimum Value             | Set minimum value that the Integer control can contain. By default, there is no minimum value.                                                                                                                       | Integer     |
| Maximum Value             | Set the maximum value that the Integer control can contain. By default, there is no maximum value.                                                                                                                   | Integer     |
| Step Size                 | Set the number the value increases or decreases when the user presses the up or down arrow keys or uses the spin control. If you set a value, the field contains the spin control.                                   | Integer     |
