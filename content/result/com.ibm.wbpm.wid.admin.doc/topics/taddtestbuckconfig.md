# Adding test bucket configurations

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. Click the Add icon . The New Configuration wizard opens.
3. In the Wizards list box, select Test
Bucket Configuration and click Next.
The Create a Test Bucket Configuration page opens.
4. In the Name field, type a name for
the new test bucket configuration.
5. In the list of component test projects, select the check
box beside each component test project, test suite, or test case that
you want to add to the test bucket configuration.
6. Click Finish. The test bucket configuration
is added to the Configurations area.
