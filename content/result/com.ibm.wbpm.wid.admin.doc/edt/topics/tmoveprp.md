# Moving properties up or down in event definitions

In the event definition editor, you can move properties
up or down in your event definitions without needing to directly modify
the underlying source code.

## About this task

## Procedure

1. In the event definition editor, select the property that
you want to move up or down.
2 Complete one of the following steps:
    - If you want to move the property up one position, right-click
the property and select Move Up. The property
moves up one position in the list of properties.
    - If you want to move the property down one position, right-click
the property and select Move Down. The property
moves down one position in the list of properties.
