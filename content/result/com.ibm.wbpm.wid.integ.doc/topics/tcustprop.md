# Defining your own properties in a custom mediation primitive

## About this task

## Procedure

1. Select the custom mediation primitive
2. Right-click it and go to Show in properties
3. In the properties view at the bottom of your window click User
properties and then click Add.
4. Type in the Name of your property
and select its Type. You can also set the Value of
your property and select whether validation is required.
