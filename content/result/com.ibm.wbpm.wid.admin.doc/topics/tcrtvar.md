# Creating test variations

## About this task

## Procedure

1. Open your test suite in the test suite editor.
2. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
3. In the Test Cases area, select the
test case for which you want to create a test variation.
4. At the bottom of the test suite editor, click the Test
Data Table tab to open the Test Data Table view.
5. Click the Add Variation icon . The New Test Variation wizard opens.
6. In the Name field, type the name
that you want to assign to the new test variation.
7 Complete one of the following steps:
    - If you want to create a new default test variation, select Create
a default test variation.
    - If you want to create a new test variation by copying the
variable values from an existing test variation, select Create
a test variation by copying from an existing one, then
select the test variation that you want to copy from in the Test
variation field.
8. Click Finish. A new tab bearing
the name of the new test variation is added to the Test Data Table
view.
