# Sorting and filtering in a business object map

## About this task

| Sorting or filtering option                                                                                                                                                 | Icon in toolbar   |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|
| To show only untransformed fields, click Show untransformed fields. Applies only to outputs.                                                                                |                   |
| To show only transformed fields, click Show transformed fields. Applies only to outputs.                                                                                    |                   |
| To show all fields, click Show all fields (default). Applies only to outputs.                                                                                               |                   |
| To sort transforms by source, click Sort transforms by source. This aligns with input fields.                                                                               |                   |
| To sort transforms by target, click Sort transforms by target (default). This aligns with output fields.                                                                    |                   |
| To sort transforms by run order, click Sort transforms by run order. This shows top-down based on run order.                                                                |                   |
| To hide all source connections, click Hide all source connections. When opening a business object map containing 50 or more transforms, Hide connections is set by default. |                   |
| To advance to the next transform, click Go to the next transform.                                                                                                           |                   |
| To go back to the previous transform, click Go to the previous transform.                                                                                                   |                   |
