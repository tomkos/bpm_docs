Adding users to a project area by using the command line

Before you begin

Be sure to have your completed configuration checklist available.

About this task

Each user can belong to only one project area in addition to the default project area. If you assign a user to a project area, the user is removed from the project area that they originally belonged to.

You can add groups to the default project area only. You can only add users individually to your nondefault project areas.

Procedure

To add users to a project area:
