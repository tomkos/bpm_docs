# Resuming component thread execution in the integration debugger

## About this task

To resume component thread execution:

## Procedure

1. In the Debug view, ensure that the correct stack frame
is selected for the component thread that you want to resume.
2. Click the Resume icon .
The component thread pauses at the next available breakpoint. If there
is no breakpoint at which any of the component threads can pause,
the component instance runs to completion and terminates.
