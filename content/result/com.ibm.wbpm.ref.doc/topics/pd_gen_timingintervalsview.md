# TIMINGINTERVALS view (deprecated)

The TIMINGINTERVALS view includes the following columns:

| Column             | Description                                                                                                                                             |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| TIMING\_INTERVAL\_ID | The primary key of this timing interval.                                                                                                                |
| NAME               | The name assigned to the timing interval in the Process Designer. For autotracking timing intervals, this name corresponds to the name of the activity. |
| DESCRIPTION        | The description of the timing interval (if one was provided in the Process Designer).                                                                   |
