Class HealthCondition

  • java.lang.Object

    • com.ibm.casemgmt.api.HealthCondition

  • public class HealthCondition extends java.lang.Object Represents a HealthCondition instance that is associated with a case. Instances of HealthCondition are obtained by calling the fetchCaseHealthConditions method on a Case instance, which returns a list of HealthCondition instances in the case.

    HealthCondition objects are stored in the Content Engine repository as objects of CmAcmHealthCondition class.
