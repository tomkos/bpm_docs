## Theme definitions

The design mode of the theme editor contains a simulation of this view. If you hover over the
simulation, it lists the theme definitions that determine the appearance of the view. For
information on the theme editor, see Creating themes.

## Configuration properties

<!-- image -->

| Configuration property       | Description                                                                                                                                                                                                                                   | Data type   |
|------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------|
| Style > Show border          | Select this option to add a visible border to the tabs.                                                                                                                                                                                       | Boolean     |
| Style > Selected tab color   | Set the color for the tab that is currently open. In the Classic theme in the System Data toolkit, the colors are: Dark blue for primary (default) White for alternate Light blue for info Green for success Yellow for warning Red for alert | String      |
| Style > Unselected tab color | Set the color for the tabs that are not open.The default value is Alternate.                                                                                                                                                                  | String      |
