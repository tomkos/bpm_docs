# Specifying a default data pool

## About this task

## Procedure

1. Click Window > Preferences to open the Preferences window.
2. In the Preferences window, expand Business Integration and
select Integration Test Client. The Integration
Test Client page opens.
3 In the Default data pool field ofthe Data Pool group, complete one of the followingsteps:
    - If you want to select an existing data pool as the default
data pool, select the name of the data pool from the list of available
data pools or select Browse to locate and select
a data pool.
    - If you want to create a new data pool as your default data
pool, select New. Information about creating
a new data pool is found in the topic "Adding data pools."
4. Click OK.
