# Defining tooltips

## Procedure

- ${dataPoint.x}: Substitutes the X-axis
value of the hovering data point
- ${dataPoint.y}: Substitutes the Y-axis
value of the hovering data point
- ${dataPoint.labels.x}: Substitutes the
X-axis label of the hovering data point
- ${series.0.name}: Substitutes the first
data series name in the associated plot
- ${series.1.name}: Substitutes the second
data series name (if applicable) in the associated
plot
