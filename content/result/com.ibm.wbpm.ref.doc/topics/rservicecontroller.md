# Service Controller

To automatically refresh the controls in the view, see Refresh Controller control. To manually refresh the controls
in the view, see Refresh Button control.

## Restrictions and limitations

None

## Data binding

| Description                                                                                                                          | Data type   |
|--------------------------------------------------------------------------------------------------------------------------------------|-------------|
| Determines whether boundary events are broadcast to other controls in the view, for example, so that the variables can be refreshed. | Boolean     |
