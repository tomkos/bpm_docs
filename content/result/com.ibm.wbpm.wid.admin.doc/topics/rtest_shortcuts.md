# Keyboard shortcuts for the integration test client

Many of these keyboard shortcuts can be invoked from menus,
which you can open by pressing Shift+F10.

Keyboard
shortcuts associated with the integration test client are available
in the pages, wizards, editors, window boxes, and views:

- Business Integration view
- Preferences window
- Dialogs
    - Add Element window
    - Value Name window
- Editors

- Assembly editor
- Data pool editor
- Programmatic emulation editor
- Value editor
- Pages

- Configurations page
- Events page
- Wizards

- New Emulator wizard
- New Monitor wizard
- New Programmatic Emulator wizard
- New Test Configuration wizard
- New Test Module wizard
- Select Deployment Location wizard
- Select Deployment Location per Module wizard

The shortcut keys for the integration test client are described
in the following sections.

## Add Element window

The following
table describes the integration test client shortcut keys that you
can invoke in the Add Element window:

| Key combination   | Function                                                              |
|-------------------|-----------------------------------------------------------------------|
| Alt+M             | Moves the focus to the Select the module that defines the type field. |
| Alt+N             | Moves the focus to the Enter the number of new elements to add field. |

This feature is especially useful in the value and data pool editors, where you can use the Shift+Tab key combination to move the focus to a row in the editor and then use the Enter key to put the editor in edit mode. Once the editor is in edit mode, you can use the Tab key to move the focus backwards from one value to the next in the editor. |
