# Creating relationships between process instances by using REST
API

- To create or remove a relationship between two process instances,
use the Process Instance Relationship resource.
- To retrieve a process instance that is dependent on the current
process instance, use the Process Instance Dependent Process resource.
- To retrieve a list of process instances that the current process
instance depends on, use the Process Instance Depended On Processes resource.
- To retrieve a list of process instances that have independent
relationships with the current process instance, use the Process Instance Related Processes resource.
- To retrieve a list of relationships that are associated with the
process instance, use the Process Instance Relationships resource.
- To update and delete relationships, use the Relationship resource.
