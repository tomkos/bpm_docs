# Removing test cases from a test bucket configuration

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test bucket configuration.
3. Select one or more test suites or test cases that you want
to remove.
4. Click the Remove icon . The selected test suites and test cases
are removed from the test bucket configuration.
