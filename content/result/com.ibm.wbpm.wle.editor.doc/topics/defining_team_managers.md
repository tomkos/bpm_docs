# Defining team managers

## Before you begin

## Procedure

For statically defined teams, complete the following
actions.

1. If you want to define a new team of managers, complete
the actions in Creating a team.
2 If you want to add or remove managers from an existing team of managers, complete thefollowing actions using the designer.
    1. Open the team of managers that you want to modify.
    2. Add or remove users and groups in the members list.
    3. Optional: To modify the team that manages this team of managers, use the
Managers section. You can either create a team, select an existing team, or
delete the currently selected team of managers.
3. Click Save or Finish Editing.
