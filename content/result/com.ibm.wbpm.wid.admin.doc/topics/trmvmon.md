# Removing monitors from a test configuration module

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration,
module, and Monitors containers.
3. In the Monitors container, select
one or more monitors that you want to remove.
4. Click the Remove icon . The selected monitors are removed from
the test configuration module.
