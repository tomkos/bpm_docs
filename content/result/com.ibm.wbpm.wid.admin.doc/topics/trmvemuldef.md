# Removing emulator definition steps

## About this task

## Procedure

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select one
or more emulator definition steps that you want to remove, such as
emulator definition steps for components, references, and inline and
stand-alone human steps.
3. At the top of the Test Cases area, click the Remove icon . The selected emulator definition steps
are removed from the Test Cases area.
