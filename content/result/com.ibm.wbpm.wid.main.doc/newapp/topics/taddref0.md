# Adding more than one wire to a partner reference

To add more than one wire to a partner reference, you must
change the multiplicity setting for the reference to 0..n.

## About this task

## Procedure

To change this setting, complete the following steps:

1. In the assembly editor, click the component or stand-alone
references node so that it is selected on the canvas.
2. In the Properties view, select Details to display the properties of the partner references.
3. In the tree view, select the partner reference that you
want to modify. You can see its properties in the pages in the right
frame.
4. In the Details page, select 0..n for Multiplicity.
