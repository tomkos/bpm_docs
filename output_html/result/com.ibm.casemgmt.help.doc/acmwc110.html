Search with stored search event

| Description | Run the stored search that is specified in the payload and display the list of documents that are returned by the search. | |---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------| | Event ID | icm.ReceivedStoredSearch | | Payload | StoredSearch.objectStoreName The IBM Content Navigator repository ID of the target object store or the target object store name.
