# Adding values to data pools

## About this task

## Procedure

1. In the integration test client, right-click a value in
the value editor and select Add Value to Pool.
The Value Name window opens. (Note, however, that if no data pool
yet exists or only the pre-6.1.2 global data pool exists, the Value
Name window will not open immediately. Instead, the New Data Pool
wizard will automatically open to enable you to create a new data
pool. After you have created a data pool, the Value Name window will
open. Information about creating a new data pool is found in the topic
"Adding data pools.")
2. In the Enter the name of the value to add to
the pool field, either accept the current value name or
type a new name for the value.
3 In the Data pool field, completeone of the following steps:
    - If you want to add the value to an existing data pool, either
accept the data pool that is currently specified or select another
data pool from the list of recently used data pools. Alternatively,
you can select Browse to locate and select
any data pool from your workspace.
    - If you want to add the value to a new data pool, select New.
Information about creating a new data pool is found in the topic "Adding
data pools."
4. Click OK.
