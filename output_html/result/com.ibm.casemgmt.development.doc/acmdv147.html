Case stage

Case stage performs some actions like completing current stage, holding the current stage, releasing the current stage, and starting previous stage. You can use the Case stages REST supported methods: GET and POST.

  • GET method for Case stage

The GET method can be used to retrieve all stages' information or current stage for a given case instance. - POST method for Case stage

The POST method can perform case stage actions like completing current stage, holding the current stage, releasing the current stage, and starting previous stage.
