# Filtering events

## About this task

## Procedure

1. In the integration test client, click the Events tab
to open the Events page.
2. At the top of the Events page, click the Filters icon . The Event Filter window opens.
3. In the Select the filters to apply list,
select the check boxes of those events that you want to filter out.
4. Click OK. The events that you selected
will not be displayed in the Events page.
