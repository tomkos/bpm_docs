# Working with comma-separated values (CSV) data

You can either import a pre-populated spreadsheet in a comma-separated
values (.csv) file, or you can export existing relationship instance
data to a .csv file.

The following topics provide the concepts and step-by-step instructions
for working with CSV data:

Using the relationship editor in IBM Integration Designer, you can export relationship instance data into a .csv file, such as an Excel spreadsheet.
