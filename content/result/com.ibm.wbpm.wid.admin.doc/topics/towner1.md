# Specifying a potential owner for a human task

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration,
module, and Human Task Emulators container.
3. Select the human task emulator for which you want to specify
a potential owner for the human task.
4 In the Detailed Properties area,complete the following tasks:
    1. In the User ID field, type the
name of the potential owner that you want to use to claim the human
task.
    2. In the Password field, type the
password for the user ID of the potential owner.
