# Synchronizing a subflow instance and implementation

## About this task

## Procedure

1. Switch to the Problems view.
2. Select the error.
3. Right-click, and select Quick fix.
4. The Quick fix window opens, showing all the subflow instances
that do not match the subflow implementation.
5. Ensure that all the subflow instances are selected, and
click Finish.  This action will
synchronize all of the subflow instances with the subflow implementation.
