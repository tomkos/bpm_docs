# Selecting a calendar type for the completion conditions

## Before you begin

## About this task

## Procedure

1. Change the focus from the people assignment to the human
task itself. In the Human Task editor click the Name field
at the top. This shifts the focus from the people assignment to the
human task allowing you to set task level options.Be
aware that this changes impacts all duration definitions for this
task.
2. Select the Duration tab in the Properties
view.
3. Choose the type of calendar you want to use in the Calendar
type field.
4. Optional: Save the task. Press Ctrl+S.
It is good practice to routinely save your work.
5. Return the focus to the people assignment. In
the Human Task editor click Potential Owners.
On the Completion tab of the Properties
view you will see that the Calendar type has
changed.
