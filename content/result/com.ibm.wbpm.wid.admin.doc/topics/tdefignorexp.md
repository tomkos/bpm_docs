# Ignoring exception errors for an invocation

## About this task

## Procedure

To ignore exceptions for the current invocation, complete
the following steps:

1. In the test suite editor, click the Test Cases tab
to open the Test Cases page.
2. In the Test Cases area, select the
invocation where you want to ignore the exceptions.
3. In the Detailed Properties area, select the Ignore
exceptions check box.
