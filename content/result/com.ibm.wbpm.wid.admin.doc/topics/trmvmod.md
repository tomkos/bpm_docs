# Removing modules from a test configuration

## About this task

## Procedure

1. In the integration test client, click the Configurations tab.
The Configurations page opens.
2. In the Configurations area, expand the test configuration.
3. If there is more than one module listed under the test
configuration, select one or more modules that you want to remove.
4. Click the Remove icon . The selected modules are removed from the
test configuration.
