Class HealthAlert

  • java.lang.Object

    • com.ibm.casemgmt.api.HealthAlert

  • public class HealthAlert extends java.lang.Object Represents a JSON object that is stored in the CmAcmHealthDetails property of the Health Condition object. Instances of HealthAlert are obtained by calling the getHealthDetails method on a HealthCondition instance, which returns a list of HealthAlert objects.

    The HealthAlert object is used for message localization purpose.
