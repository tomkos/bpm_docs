GET method for the list of case types resource

URI

/CASEREST/v1/solution/{solution name}/casetypes

| Name | Type | Description | |-----------------|--------|------------------------------------------------------------------------------| | {solution name} | String | The name of the solution for which the list of case types is to be returned. |

| Name | Type | Required? | Description | |-------------------|--------|-------------|--------------------------------------------------------------------| | TargetObjectStore | String | Yes | The symbolic name of the object store that contains the case type. |

Request content

The request for this method contains no JSON content.
