Unexpected results when you use a number spinner editor with a property of type float

  • Specify an initial value for the property, either in the model layer or by setting the default value in Case Builder.
  • Select the Round automatically option.
  • Specify the number of decimal places.
  • Specify reasonable values, such as 1 or 0.25, for the Small increment and Large increment fields. (Setting these fields to values such as 0.333 can cause problems.)
