Class CaseMgmtUpdatingBatch

  • java.lang.Object

    • com.ibm.casemgmt.api.CaseMgmtUpdatingBatch

  • public class CaseMgmtUpdatingBatch extends java.lang.Object Submits changes to Case Management objects in a batch. One round-trip to the server is made to submit the changes for multiple objects. This class directly handles the changes for Case, Task and ExternalDocumentReference Case Java API objects.

    Internally, an UpdatingBatch instance from the IBM Content Engine Java API is managed, collecting each change to be submitted to the server.
