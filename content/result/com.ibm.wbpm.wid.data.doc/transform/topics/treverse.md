# Creating a reverse map

## About this task

To use the reverse map function in the business object
map editor, follow these steps:

## Procedure

1. From the business object map editor, open the business
object map to create a reverse map.
2. Click Create reverse map from the
toolbar:
3. The New Reverse Business Object Map window opens. Complete
the wizard. The new reverse map opens in the business
object map editor.
