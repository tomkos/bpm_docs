# Creating transforms using arrays

## About this task

<!-- image -->

To select the check box so that the business object map
editor interprets dots as an array index and not as part of the field
name, follow these steps:

## Procedure

1. From the Business Integration perspective, click Window > Preferences.
2. Expand Business Integration in the list and select Business
Object Map Editor.
3. Select the Tolerate deprecated zero base style
of array indexing check box and click OK.
