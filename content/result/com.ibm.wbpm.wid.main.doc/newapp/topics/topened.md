# Opening integration solution diagrams

When you create an integration solution, an integration
solution diagram is automatically added that you can open in the integration
solution editor.

## About this task

## Procedure

1. Switch to the Business Integration view.
2 In the Integration Solutions section,choose from one of the following options: The integration solution editor opens.
    - Right-click your integration solution and select Open
Solution Diagram.
    - Double-click Solution Diagram. (Alternatively,
you can right-click Solution Diagram and select Open.)
